C: CSS

Vad är en regel i CSS och vad består den av?

Svar: CSS regler består av två olika delar. De här delerna anger vilka element som ska påverkas och den andra vilka egenskaper som man ska ändra på. Selektorn väljer sedan ut specifika elemt och tar ett beslut om vilka element som ska påverkas av själva CSS regeln.
